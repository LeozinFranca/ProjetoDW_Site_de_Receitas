# ProjetoDW_Site_de_Receitas
Projeto de Desenvolvimento Web com o objeto da criação de um site

 Neste Projeto desenvolveremos um site com o intuito de armazenar receitas de alimentos, localização de restaurantes e outros servicos relacionados a área da alimentação perto da região de Euclides da Cunha, como também haverá algumas sujestões de alimentos para aqueles que buscam uma alternativa saudavel de alimentação. 
 Haverá abas exclusivas para as receitas, para os restaurantes onde será possivel se encontrar links para o acesso das redes pessoais dos mesmos e para sujestões de alimentos mais saudáveis.(as abas comunicam-se uma com a outra)
